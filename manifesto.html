Training Session Structure

  1. Warm-up — Team lap + stretching (top & bottom half), then 5–10 min individual stretch
  2. Phase 1 — Drilling (20–25 min) — Fundamentals, in-system drills (e.g., throw → set → finish)
  3. Phase 2 — Advanced drilling (20–25 min) — Full rotations (first touch → set → attack)
  4. Game forms / Pickup (remaining time) — Coach picks nets, mixing skill levels. Use modified scoring to reinforce the day's focus (e.g., push = 2 pts, same server 5x in a row)

Daily Pickups (16:30–18:00)

  • All groups start together at 16:30 — everyone plays with everyone, all fields
  • Game 1: 15 minutes, random teams across all nets
  • Game 2+: Winners move one net up, losers stay. Old teammates split via rock-paper-scissors to form new teams
  • Format: standard roundnet
  • Repeat rotation for remaining time
